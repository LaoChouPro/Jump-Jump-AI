# 跳一跳AI数据收集工具使用说明

## 数据采集工具新特性

### 全局鼠标选择区域
- 点击"选择区域"后，可以在**任意应用窗口**上直接点击选择截图区域
- 无需切换到特定窗口，直接在游戏界面上操作
- 按钮会变为"停止选择"，可随时取消操作

### 操作流程
1. 启动工具：`python 启动数据采集.py`
2. 打开跳一跳游戏
3. 点击"选择区域"进入选择模式
4. 直接在游戏窗口上：
   - 点击游戏区域左上角
   - 点击游戏区域右下角
5. 选择完成后会自动开启实时预览视频画面
6. 可以调整预览帧率（1-30 FPS）
7. 观察实时画面，确认截图区域正确
8. 启用快捷键或点击"截图"按钮采集数据

### 快捷键功能 🆕
- **启用快捷键**: 勾选"启用快捷键"选项
- **快捷键组合**: `Cmd+Opt+E` (macOS) 或 `Ctrl+Alt+E` (其他系统)
- **快速截图**: 无需切换窗口，直接按快捷键即可截图
- **状态反馈**: 快捷键截图后会显示完成提示

### 实时预览功能
- **自动开启**：区域选择完成后自动开启实时预览
- **手动控制**：可通过"开启预览"/"关闭预览"按钮控制
- **帧率调节**：支持1-30 FPS帧率调整，平衡性能和实时性
- **视频流**：连续显示选定区域的实时画面

### 权限要求
**macOS用户必须授权**：
- 前往"系统偏好设置 > 安全性与隐私 > 辅助功能"
- 添加Terminal或Python到允许列表
- 或者在首次运行时根据系统提示授权

**权限检查**：
- 运行 `python tools/权限检查.py` 预先检测系统权限
- 如果权限检查失败，请按提示配置权限后重试

### 采集建议
- 建议采集不同难度场景的截图
- 每个场景采集20-50张图片
- 包含不同的方块形状和颜色
- 确保小人在不同位置的截图

## 标注工具使用

### 界面布局 🆕
- **上下分割布局**: 上方控制栏，下方大图片显示区域
- **水平控制栏**: 文件控制、类别选择、标注管理横向排列
- **更大显示区域**: 图片显示区域更大，标注更精确

### 快速标注模式 🆕
- **启用方式**: 勾选"快速标注模式"
- **智能引导**: 自动按顺序提示标注类别 (小人 → 小人方块 → 普通方块)
- **自动切换**: 每完成一个标注自动切换到下一个类别
- **当前提示**: "→ 请标注: 小人" 实时显示当前应该标注的类别
- **自动流程**: 标注完3个区域后自动保存并跳转下一张
- **效率提升**: 无需手动选择类别、保存和翻页

### 标注类别
- **小人** (红色框): 游戏角色本体
- **小人所在方块** (绿色框): 当前站立平台
- **普通方块** (蓝色框): 目标跳跃平台

### 标注技巧
- 框选时尽量贴合目标边界
- 小人建议框选整个身体轮廓
- 方块建议框选完整的可见区域
- 标注框线条加粗，更易识别

### 快捷操作
- 双击标注列表删除标注
- Del键删除选中标注
- "跳转未标注"按钮快速定位未完成图片
- 自动跳转到第一个未标注图片
